# Newton's-Apples 

Um jogo simples feito com HTML, CSS e JavaScript usando Canvas.

---
## História

O jogo é inspirado na famosa história de **Isaac Newton** e a ideia da gravidade.

Segundo a lenda, Newton teria observado uma maçã caindo de uma árvore, o que o levou a refletir sobre a força da gravidade. A partir disso, surgiu uma das teorias mais importantes da física.

Neste jogo, você vive uma versão divertida dessa ideia, tentando **capturar as maçãs antes que elas caiam**, como se estivesse testando as leis da gravidade na prática.

---

## Como jogar

- Use as teclas **A / D** ou **setas** para mover o personagem
- No celular, toque na esquerda ou direita da tela para se mover
- O objetivo é pegar as maçãs que caem
- Cada maçã coletada vale pontos
- Se uma maçã cair no chão, você perde vida

---
## Regras

- Ganhe pontos ao coletar maçãs 
- Perde pontos/vidas se deixar elas caírem
- O jogo termina quando suas vidas acabam

---
## Tecnologias usadas

- HTML5 Canvas
- JavaScript puro
- CSS
- Node.js (backend)
- Supabase (banco de dados)
- Render (deploy do servidor)

---
## Backend

O jogo usa um backend em **Node.js** hospedado na plataforma **Render**, responsável por:
- Receber a pontuação dos jogadores
- Salvar os dados no banco **Supabase**
---
## Responsivo

O jogo funciona em:
- Desktop 
- Celular 

---
## Sistema de pontuação

- +100 pontos por maçã coletada
- -60 pontos por maçã perdida
